# Doxygen Unity Template

A ready-to-use template for integrating **Doxygen** documentation generation into **Unity** projects - including support for Tooltip-to-XML conversion and automated documentation publishing via **GitHub Actions**.

## Overview

This repository provides a complete setup to bring high-quality, maintainable, auto-generated documentation into your Unity workflow.

It includes:

- A Unity-ready **Doxyfile**

- Tooltip-to-XML conversion filters (using either `python` or `bash`)

- A GitHub Actions workflow for **fully automated documentation builds**

- A [detailed wiki](https://github.com/RCL-Studio-X/Doxygen-Unity-Template/wiki) covering setup, configuration, and automation

The Unity project in this repository is created with [Unity Editor 6000.0.58f2](https://unity.com/releases/editor/whats-new/6000.0.58f2).
